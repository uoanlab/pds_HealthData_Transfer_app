# pds_HealthData_Transfer_app
ヘルスケアデータをPDSに格納するiOSアプリ

# 開発環境
- Xcode version 10.1  
- Swift version 4.2.1  

# 実行環境
- iOS, version 12以降
- [Personium（オープンソースPDS）](https://personium.io/ja/index.html)
